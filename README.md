This Weather App project uses an API to get the real-time weather data and displays it in a stylish way with good UI, with many features such as:

--Image Display of Weather
--Temperature Display
--Humidity, Air Pressure, Feels Like Temperature, Wind Direction, etc

In the project, you have to enter the name of any city and all details will be displayed.

Many more features were added to this project like-->

--Keyboard Typing sound
--Effects
